# Acciones del menú contextual / atajos

- Click derecho sobre un nodo abre menú:
  - **Agregar subtema** (`add-child`) – inserta un hijo directo y reordena el layout.
  - **Agregar tema hermano** (`add-sibling`) – crea otro nodo al mismo nivel; si existía padre lo usa, si no crea un tema nuevo.
  - **Agregar tema flotante** – nodo independiente sin edges, útil para ideas sueltas.
  - **Duplicar** – copia el nodo con el mismo estilo/desc y añade “(copia)” al título.
  - **Eliminar** – borra el nodo y los edges asociados.
- Doble clic sobre un nodo entra en modo edición in-place.
- El inspector permite ajustar color de nodo, tamaño de fuente y nivel jerárquico.
